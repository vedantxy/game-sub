## 🚀 Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Modern web browser

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/vedantxy/game-sub.git
   cd game-sub
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open in browser**
   - Navigate to `http://localhost:3000`
   - Start playing!

## 🎮 How to Play

### General Tips
1. **Practice**: Each game requires practice to master
2. **Beat your high score**: Try to improve your personal best
3. **Compete**: Challenge friends and see who scores higher
4. **Learn Controls**: Take time to understand all game controls
5. **Have Fun**: Remember, the goal is to enjoy!

### Keyboard Shortcuts
- `H` - Home/Menu
- `P` - Pause
- `M` - Mute/Unmute
- `F` - Fullscreen
- `ESC` - Back/Exit

## 🤝 Contributing

Contributions are welcome! To contribute:

1. **Fork the repository**
   ```bash
   git clone https://github.com/yourusername/game-sub.git
   ```

2. **Create a feature branch**
   ```bash
   git checkout -b feature/new-game
   ```

3. **Make your changes**
   - Add new game or enhance existing ones
   - Follow code style guidelines
   - Test thoroughly

4. **Commit changes**
   ```bash
   git commit -m "Add: New game feature"
   ```

5. **Push to branch**
   ```bash
   git push origin feature/new-game
   ```

6. **Open a Pull Request**
   - Describe changes clearly
   - Link any related issues

### Code Style Guidelines
- Use meaningful variable names
- Write comments for complex logic
- Keep functions small and focused
- Follow existing code patterns
